# AI Architecture Advisor

8 specialized AI agents · 15 discovery scenarios · 3 operating modes

**Live demo:** https://danvzla.github.io/ai-architecture-advisor/

## Modes
- **Demo Mode** — No API key needed. Full pre-built Telco NOC blueprint.
- **Claude API** — Live generation via Anthropic claude-haiku-4-5
- **OpenAI API** — Live generation via gpt-4o-mini

## What it produces
Reference Architecture · Architecture Decision Records (ADRs) ·
Risk Register · 30/60/90 Roadmap · TCO/ROI/NPV Model · CFO Narrative ·
AI Readiness Scorecard

## Local development
Edit source files in `assets/`, then:
```bash
node build.js
```
Upload `dist/index.html` to GitHub as `index.html`.
